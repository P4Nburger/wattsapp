# WattsConnected (wattsapp)

macOS のメニューバーに現在の電力消費（ワット数）や電源状態を表示するシンプルなユーティリティです。IOKit を用いて AC 接続/充電中/バッテリー駆動などの状態を検出し、更新間隔をメニューから切り替えられます。

## 特徴
- メニューバーに実効ワット数を表示
- 電源状態（AC 接続/充電中/バッテリー駆動）の検出
- 更新間隔の切り替え（高頻度/標準（自動）/低頻度）
- ステータスメニューに状態の詳細（アダプタ W、実効表示など）を表示
