# WattsConnected

macOS のメニューバーに現在の電力消費（ワット数）や電源状態をリアルタイムで表示するユーティリティアプリです。

## 特徴

- ⚡ **メニューバーにワット数を表示** — 充電/AC接続/バッテリー駆動に応じた SF Symbols アイコン付き
- 🔋 **バッテリー詳細情報** — 残量(%)、健康度、サイクル数、温度、残り時間をメニューから確認
- ⚙️ **カスタマイズ可能な表示** — メニューバーの表示スタイル（アイコン＋テキスト / テキストのみ / アイコンのみ）、各項目の表示/非表示を設定で切り替え
- 🔄 **更新間隔の調整** — 高頻度（約1.5秒）/ 標準（自動）/ 低頻度（5秒）から選択可能
- 🚀 **ログイン時に自動起動**
- 💾 **設定の永続化** — すべての設定はアプリ再起動後も保持

## アーキテクチャ

SwiftUI + MVVM パターンで構成されています。

| ファイル | 役割 |
|---------|------|
| `wattsappApp.swift` | アプリエントリポイント、`MenuBarExtra` で UI 定義 |
| `PowerViewModel.swift` | バッテリー状態の監視・更新ロジック |
| `BatteryInfoProvider.swift` | IOKit を使ったバッテリー情報の取得 |
| `StatusTextFormatter.swift` | ワット数テキストのフォーマット |
| `AppSettings.swift` | `@AppStorage` によるユーザー設定の永続化 |
| `SettingsView.swift` | 設定画面（一般 / 表示 / About） |
| `AboutView.swift` | バージョン情報・リンク |
| `LaunchAtLogin.swift` | ログイン時起動の管理 |
